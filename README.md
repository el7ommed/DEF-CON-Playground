This repository is a work-in-progress playground for an ESP32 badge I picked up as a DEF CON giveaway. There’s no single goal or polished product here — I’m just exploring the hardware, trying out sketches, and seeing what the badge can do.

Expect rough experiments, snippets, and prototypes. Feel free to look around, try things on your own badge, or open an issue if you want to collaborate or ask questions.
